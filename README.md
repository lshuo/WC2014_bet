WC2014_bet
==========

World Cup 2014 bet helper
Use this on your own risk!

Environment: Python 2.7.5

Perquisite: Beautiful Soup 4

Usage:
Download match analytics webpage from https://soccer.bloombergsports.com/en/matchups/xxxxxx/, then run bet_zhihu.
Repeat the above to get bet suggestions for all matches that day before running bet_wechat.
Download wechat caipiao webpage from http://tc.w-caipiao.tenpay.com/m_wx/index.html#jczq=chooseJson, then run bet_wechat.
